# Tas3eer — Pricing Calculator for Home-Based Food Producers

Tas3eer is a simple Arabic-first web app that helps home-based food producers in Saudi Arabia price their products accurately. It calculates the real cost of a recipe (ingredients, energy, and the producer's time) and suggests a selling price based on a profit margin, with a transparent breakdown that teaches the user where every riyal comes from.

## Tech Stack
- **Frontend:** HTML, CSS, vanilla JavaScript (no frameworks, no build tools)
- **Backend & Database:** Supabase (PostgreSQL + Auth) — *planned*
- **Hosting:** Vercel — *planned*
- **Version control:** Git + GitHub

## Run Locally
This is a pure static site — no installation required.

**Option 1:** Open `index.html` directly in your browser.

**Option 2 (recommended):** Open the project folder in VS Code, install the **Live Server** extension, then right-click `index.html` → *Open with Live Server*. The page will auto-reload whenever you save a file.
